## 项目概览

### 核心价值主张
- **内容差异化**：100% AI生成原创短篇，避免版权问题
- **用户体验**：沉浸式阅读交互（上下滑动内容，左右切换小说）
- **商业模式**：视频解锁结局 + 订阅制 + 广告
- **全球化**：双语支持（繁体中文 + 英语）

### 用户画像
- **主要用户**：18-45岁，喜欢快速消费内容的读者
- **使用场景**：碎片化时间阅读（通勤、午休、睡前）
- **付费意愿**：愿意为优质内容和无广告体验付费

### 商业模式
| 收入来源 | 说明 | 预期占比 |
|---------|------|---------|
| 视频解锁 | 看广告视频解锁小说结局 | 40% |
| 订阅制 | 月/年订阅无限阅读 | 35% |
| 广告 | 信息流广告、插屏广告 | 20% |
| 打赏 | 用户打赏喜欢的小说 | 5% |

**阅读进度与解锁机制：**
```
小说总长度：3000字（约3章）
第1章（0-1000字）：完全免费
第2章（1000-2000字）：免费
第3章（2000-3000字）：
  ├─ 前2/3（2000-2666字）：免费
  └─ 后1/3（2666-3000字）：
      ├─ 看30秒视频解锁 ✓
      ├─ 订阅用户直接看 ✓
      └─ 购买单章解锁 ✓
```

**解锁弹窗设计：**
```
┌──────────────────────────┐
│  🎬 看视频解锁结局        │
├──────────────────────────┤
│                          │
│  还剩 334 字未读          │
│  看30秒广告视频即可解锁   │
│                          │
│  [看视频解锁] [订阅]     │
│                          │
└──────────────────────────┘
```

**技术实现：**
```javascript
// 阅读进度追踪
const readerState = {
  storyId: "story_001",
  currentChapter: 1,
  scrollPosition: 0,
  totalWords: 3000,
  readWords: 1500,
  isUnlocked: false,
  unlockedAt: null
}

// 解锁逻辑
function checkUnlock(readWords, totalWords) {
  const unlockThreshold = totalWords * 2/3;
  if (readWords >= unlockThreshold && !isUnlocked) {
    showUnlockModal();
  }
}
```

### 3. 响应式设计

**断点设置：**
- 手机 (320px - 768px)：主要目标
- 平板 (768px - 1024px)：支持
- 桌面 (1024px+)：支持（PWA模式）

**移动优先原则：**
- 所有设计从手机开始
- 逐步增强到平板/桌面
- 触摸友好的交互（最小点击区域 44x44px）

### 4. 性能优化

**关键指标：**
- FCP (First Contentful Paint): < 1.5s
- LCP (Largest Contentful Paint): < 2.5s
- CLS (Cumulative Layout Shift): < 0.1

**优化策略：**
```javascript
// 1. 图片懒加载
<img v-lazy="story.cover" />

// 2. 虚拟滚动（长列表）
<virtual-scroller :items="stories" />

// 3. 代码分割
const Reader = () => import('./pages/Reader.vue')

// 4. 预加载下一篇小说
prefetchStory(nextStoryId)

// 5. Service Worker 缓存
// 缓存首页、常用资源、已读小说内容
```

### 2. 数据库设计

#### 2.1 核心表结构

**users 表**
```sql
CREATE TABLE users (
  id BIGSERIAL PRIMARY KEY,
  email VARCHAR(255) UNIQUE NOT NULL,
  phone VARCHAR(20),
  password_hash VARCHAR(255) NOT NULL,
  username VARCHAR(100) UNIQUE NOT NULL,
  avatar_url TEXT,
  language VARCHAR(10) DEFAULT 'zh-TW',
  subscription_tier VARCHAR(50) DEFAULT 'free', -- free, premium, vip
  subscription_expires_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  deleted_at TIMESTAMP,
  
  INDEX idx_email (email),
  INDEX idx_username (username),
  INDEX idx_subscription_tier (subscription_tier)
);
```

**stories 表**
```sql
CREATE TABLE stories (
  id BIGSERIAL PRIMARY KEY,
  title VARCHAR(255) NOT NULL,
  slug VARCHAR(255) UNIQUE NOT NULL,
  category VARCHAR(50) NOT NULL, -- 悬疑, 爱情, 奇幻, 都市
  language VARCHAR(10) NOT NULL, -- zh-TW, en
  content TEXT NOT NULL, -- 完整内容
  summary TEXT NOT NULL,
  cover_url TEXT,
  author_name VARCHAR(100) DEFAULT 'AI-StoryFlow',
  word_count INT NOT NULL,
  chapter_count INT DEFAULT 1,
  
  -- 统计字段
  views INT DEFAULT 0,
  likes INT DEFAULT 0,
  comments_count INT DEFAULT 0,
  rating DECIMAL(3,2) DEFAULT 0,
  
  -- 内容生成相关
  ai_model VARCHAR(100), -- 使用的AI模型
  prompt_version INT, -- 提示词版本
  generated_at TIMESTAMP,
  
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  published_at TIMESTAMP,
  
  INDEX idx_category (category),
  INDEX idx_language (language),
  INDEX idx_published_at (published_at),
  INDEX idx_views (views),
  INDEX idx_rating (rating)
);
```

**story_chapters 表**
```sql
CREATE TABLE story_chapters (
  id BIGSERIAL PRIMARY KEY,
  story_id BIGINT NOT NULL REFERENCES stories(id),
  chapter_number INT NOT NULL,
  title VARCHAR(255),
  content TEXT NOT NULL,
  word_count INT,
  unlock_type VARCHAR(50) DEFAULT 'free', -- free, video, subscription, paid
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  
  UNIQUE(story_id, chapter_number),
  INDEX idx_story_id (story_id)
);
```

**reading_history 表**
```sql
CREATE TABLE reading_history (
  id BIGSERIAL PRIMARY KEY,
  user_id BIGINT NOT NULL REFERENCES users(id),
  story_id BIGINT NOT NULL REFERENCES stories(id),
  chapter_number INT,
  scroll_position INT, -- 滚动位置（字符数）
  read_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  completed_at TIMESTAMP,
  
  UNIQUE(user_id, story_id),
  INDEX idx_user_id (user_id),
  INDEX idx_read_at (read_at)
);
```

**user_unlocks 表**
```sql
CREATE TABLE user_unlocks (
  id BIGSERIAL PRIMARY KEY,
  user_id BIGINT NOT NULL REFERENCES users(id),
  story_id BIGINT NOT NULL REFERENCES stories(id),
  chapter_number INT,
  unlock_method VARCHAR(50), -- video, subscription, paid
  unlocked_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  
  UNIQUE(user_id, story_id, chapter_number),
  INDEX idx_user_id (user_id)
);
```

**user_likes 表**
```sql
CREATE TABLE user_likes (
  id BIGSERIAL PRIMARY KEY,
  user_id BIGINT NOT NULL REFERENCES users(id),
  story_id BIGINT NOT NULL REFERENCES stories(id),
  liked_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  
  UNIQUE(user_id, story_id),
  INDEX idx_user_id (user_id)
);
```

**video_views 表**
```sql
CREATE TABLE video_views (
  id BIGSERIAL PRIMARY KEY,
  user_id BIGINT NOT NULL REFERENCES users(id),
  story_id BIGINT NOT NULL REFERENCES stories(id),
  video_id VARCHAR(255), -- 广告视频ID
  viewed_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  completed BOOLEAN DEFAULT FALSE,
  
  INDEX idx_user_id (user_id),
  INDEX idx_viewed_at (viewed_at)
);
```

**subscriptions 表**
```sql
CREATE TABLE subscriptions (
  id BIGSERIAL PRIMARY KEY,
  user_id BIGINT NOT NULL REFERENCES users(id),
  tier VARCHAR(50) NOT NULL, -- premium, vip
  price DECIMAL(10,2),
  currency VARCHAR(10) DEFAULT 'USD',
  billing_cycle VARCHAR(50), -- monthly, yearly
  started_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  expires_at TIMESTAMP NOT NULL,
  auto_renew BOOLEAN DEFAULT TRUE,
  payment_method VARCHAR(50),
  
  INDEX idx_user_id (user_id),
  INDEX idx_expires_at (expires_at)
);
```

**analytics 表**
```sql
CREATE TABLE analytics (
  id BIGSERIAL PRIMARY KEY,
  date DATE NOT NULL,
  metric_type VARCHAR(100), -- daily_users, story_views, revenue
  metric_value DECIMAL(15,2),
  story_id BIGINT REFERENCES stories(id),

### 4. 内容分段逻辑

**自动分章算法：**
```python
def auto_segment_story(content, target_chapters=3):
    """
    根据内容自动分段成多个章节
    """
    # 1. 按段落分割
    paragraphs = content.split('\n\n')
    
    # 2. 计算每个段落的字数
    para_lengths = [len(p) for p in paragraphs]
    
    # 3. 计算目标章节字数
    total_words = sum(para_lengths)
    target_words_per_chapter = total_words / target_chapters
    
    # 4. 贪心算法分配章节
    chapters = []
    current_chapter = []
    current_words = 0
    
    for i, para in enumerate(paragraphs):
        current_chapter.append(para)
        current_words += para_lengths[i]
        
        # 如果当前章节接近目标字数，且不是最后一段
        if current_words >= target_words_per_chapter and i < len(paragraphs) - 1:
            chapters.append('\n\n'.join(current_chapter))
            current_chapter = []
            current_words = 0
    
    # 5. 添加最后一章
    if current_chapter:
        chapters.append('\n\n'.join(current_chapter))
    
    return chapters
```

### 5. 质量检查

**自动审核规则：**
```python
def quality_check(story):
    """
    检查生成的故事是否符合质量标准
    """
    issues = []
    
    # 1. 字数检查
    word_count = len(story['content'])
    if word_count < 1500 or word_count > 5000:
        issues.append(f"字数不符: {word_count}")
    
    # 2. 重复检查（检测是否过度重复）
    sentences = story['content'].split('。')
    unique_sentences = len(set(sentences))
    if unique_sentences / len(sentences) < 0.7:
        issues.append("内容重复度过高")
    
    # 3. 敏感词检查
    sensitive_words = ['政治敏感词', '不当内容']
    for word in sensitive_words:
        if word in story['content']:
            issues.append(f"包含敏感词: {word}")
    
    # 4. 故事完整性检查
    if not story.get('title') or not story.get('summary'):
        issues.append("缺少标题或简介")
    
    # 5. 语言质量检查
    if has_grammar_errors(story['content']):
        issues.append("存在语法错误")
    
    return {
        'passed': len(issues) == 0,
        'issues': issues,
        'score': max(0, 100 - len(issues) * 10)
    }
```

import anthropic
import json
from datetime import datetime

class StoryGenerator:
    def __init__(self):
        self.client = anthropic.Anthropic()
        self.model = "claude-3-5-sonnet-20241022"
    
    def generate_story(self, config):
        """
        生成短篇小说
        """
        prompt = self._build_prompt(config)
        
        message = self.client.messages.create(
            model=self.model,
            max_tokens=4000,
            temperature=config.get('temperature', 0.8),
            messages=[
                {
                    "role": "user",
                    "content": prompt
                }
            ]
        )
        
        raw_content = message.content[0].text
        story = self._parse_story(raw_content, config)
        
        # 质量检查
        quality = self._quality_check(story)
        if not quality['passed']:
            raise ValueError(f"质量检查失败: {quality['issues']}")
        
        return story
    
    def _build_prompt(self, config):
        """构建提示词"""
        return f"""你是一个创意短篇小说作家。请根据以下要求生成一篇完全原创的短篇小说：

【题材】{config['genre']}
【语言】{config['language']}
【字数】{config['word_count']}字
【风格】{config['style']}

【故事要素】
- 主人公：{config['protagonist']}
- 冲突：{config['conflict']}
- 高潮：{config['climax']}
- 结局类型：{config['ending_type']}

【创意要求】
1. 故事必须完全原创，不能参考或模仿任何已知作品
2. 包含意外的转折或反转
3. 人物性格鲜明，对话自然
4. 节奏紧凑，引人入胜
5. 结局有深意或启发

【输出格式】
请按以下JSON格式输出：
{{
  "title": "故事标题",
  "summary": "50字以内的简介",
  "content": "完整故事内容"
}}

现在开始创作："""
    
    def _parse_story(self, raw_content, config):
        """解析AI生成的内容"""
        try:
            # 尝试提取JSON
            json_start = raw_content.find('{')
            json_end = raw_content.rfind('}') + 1
            json_str = raw_content[json_start:json_end]
            story_data = json.loads(json_str)
        except:
            # 降级处理
            story_data = {
                'title': '未命名故事',
                'summary': raw_content[:100],
                'content': raw_content
            }
        
        # 自动分段
        chapters = self._auto_segment(story_data['content'])
        
        return {
            'title': story_data.get('title', '未命名故事'),
            'summary': story_data.get('summary', ''),
            'content': story_data.get('content', ''),
            'chapters': chapters,
            'word_count': len(story_data.get('content', '')),
            'category': config['genre'],
            'language': config['language'],
            'generated_at': datetime.now().isoformat()
        }
    
    def _auto_segment(self, content, target_chapters=3):
        """自动分段"""
        paragraphs = content.split('\n\n')
        para_lengths = [len(p) for p in paragraphs]
        total_words = sum(para_lengths)
        target_words_per_chapter = total_words / target_chapters
        
        chapters = []
        current_chapter = []
        current_words = 0
        
        for i, para in enumerate(paragraphs):
            current_chapter.append(para)
            current_words += para_lengths[i]
            
            if current_words >= target_words_per_chapter and i < len(paragraphs) - 1:
                chapters.append({
                    'number': len(chapters) + 1,
                    'content': '\n\n'.join(current_chapter),
                    'word_count': current_words
                })
                current_chapter = []
                current_words = 0
        
        if current_chapter:
            chapters.append({
                'number': len(chapters) + 1,
                'content': '\n\n'.join(current_chapter),
                'word_count': current_words
            })
        
        return chapters
    
    def _quality_check(self, story):
        """质量检查"""
        issues = []
        
        if story['word_count'] < 1500:
            issues.append("字数过少")
        
        if not story['title'] or story['title'] == '未命名故事':
            issues.append("缺少标题")
        
        return {
            'passed': len(issues) == 0,
            'issues': issues
        }

```typescript
import { defineStore } from 'pinia';
import { ref } from 'vue';
import { api } from '../api/client';

export const useStoryStore = defineStore('story', () => {
  const stories = ref([]);
  const currentStory = ref(null);
  const loading = ref(false);

  const fetchStories = async (params: any) => {
    loading.value = true;
    try {
      const response = await api.get('/stories', { params });
      stories.value = response.data.items;
      return response.data;
    } finally {
      loading.value = false;
    }
  };

  const fetchStory = async (id: string) => {
    loading.value = true;
    try {
      const response = await api.get(`/stories/${id}`);
      currentStory.value = response.data;
      return response.data;
    } finally {
      loading.value = false;
    }
  };

  const saveProgress = async (storyId: string, progress: any) => {
    await api.post('/reading-history', {
      story_id: storyId,
      ...progress
    });
  };

  const unlockWithVideo = async (storyId: string) => {
    const response = await api.post(`/stories/${storyId}/unlock`, {
      unlock_method: 'video'
    });
    return response.data;
  };

  return {
    stories,
    currentStory,
    loading,
    fetchStories,
    fetchStory,
    saveProgress,
    unlockWithVideo
  };
});

```python
from fastapi import FastAPI, HTTPException
from pydantic import BaseModel
from generator import StoryGenerator
from database import Database
import logging

app = FastAPI()
generator = StoryGenerator()
db = Database()

logger = logging.getLogger(__name__)

class GenerationRequest(BaseModel):
    genre: str
    language: str
    word_count: int
    style: str
    protagonist: str
    conflict: str
    climax: str
    ending_type: str

@app.post("/generate")
async def generate_story(request: GenerationRequest):
    """生成短篇小说"""
    try:
        config = request.dict()
        story = generator.generate_story(config)
        
        # 保存到数据库
        story_id = db.save_story(story)
        
        return {
            "code": 0,
            "data": {
                "story_id": story_id,
                "title": story['title'],
                "word_count": story['word_count']
            }
        }
    except Exception as e:
        logger.error(f"Generation failed: {str(e)}")
        raise HTTPException(status_code=500, detail=str(e))

@app.get("/health")
async def health_check():
    """健康检查"""
    return {"status": "ok"}

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="0.0.0.0", port=8000)
